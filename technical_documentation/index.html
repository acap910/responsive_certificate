Introduction

Python is a high-level, interpreted programming language known for its readability and versatility.

Created by Guido van Rossum and first released in 1991, Python has become one of the most popular programming languages worldwide.

print("Hello, World!")

Key features include:

  • Easy-to-read syntax
  • Cross-platform compatibility
  • Large standard library
Getting Started

To begin programming in Python, you'll need to install it on your computer.

Follow these steps:

  1. Download Python from the official website
  2. Run the installer
  3. Verify installation by running python --version in your terminal
# This is a single-line comment in Python

Once installed, you can run Python code in:

  • The interactive interpreter
  • Script files (.py extension)
Variables and Data Types

Python has several built-in data types:

x = 5 # Integer y = "Hello" # String z = 3.14 # Float

Variables are created when you first assign a value to them:

counter = 100 # An integer assignment miles = 1000.0 # A floating point name = "John" # A string

Python is dynamically typed, meaning you don't need to declare variable types.

Control Flow

Control flow statements determine the order in which code executes.

Conditional statements:

if x > 0: print("Positive") elif x == 0: print("Zero") else: print("Negative")

Loop statements:

# While loop while x < 5: print(x) x += 1 # For loop for i in range(5): print(i)
Functions

Functions are reusable blocks of code that perform a specific task.

def greet(name): """This function greets the person passed in""" print("Hello, " + name + ". Good morning!")

Function components:

  • def keyword
  • Function name
  • Parameters (optional)
  • Docstring (optional)
  • Function body
  • return statement (optional)

Example with return value:

def square(number): return number * number
